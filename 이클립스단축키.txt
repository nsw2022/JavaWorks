소스편집
Ctrl + I : 들여쓰기 자동 수정
Ctrl + Y : 소스 수정 취소의 취소

위치 이동
Ctrl + 특정 클래스 마우스 왼쪽 클릭(F3) : 특정 클래스로 이동

Ctrl + O : 소스의 메소드 리스트 확인

Ctrl + L : 지정한 소스 줄로 이동

Alt + 방향키(←) : 이전 작업 위치로 이동

Alt + 방향키(→) : 다음 작업 위치로 이동

Alt + 방향키(↑) : 바로위 코드 이동

Alt + 방향키(↓) : 바로위 코드 이동

F4 : 클래스명을 선택하고 F4를 누르면 해당 클래스의 상속 계층 확인

실행
Ctrl + F11 : 소스 실행 (에러가 났을 때 디버깅 하지않음)

복사
Ctrl + Alt + 방향키(↓) 한줄복사

파라미터 매개변수 확인
Ctrl + Shift + space

자동정렬
Ctrl + Shift + F 전체 자동정렬
Ctrl + i 드래그한만큼 자동정렬

----------생성자 자동완성--------------------------------
생성자 역시 마우스우클릭->Source->Generate Constructor using Fields... (필드있는 생성자,필드선택가능)

                                              ->Generate Constructor from Superclass... (기본생성자) 로 자동생성 가능하다.
